# RAG / LLM Security Scanner

A defensive security testing toolkit for Retrieval-Augmented Generation applications.

This project will include:

- A vulnerable demo RAG chatbot
- A scanner engine
- Prompt injection probes
- Sensitive-data leakage detectors
- Unauthorized retrieval tests
- OWASP LLM Top 10 mapping
- JSON and HTML security reports
- A React dashboard

## Tech Stack

- Backend: Python, FastAPI
- Frontend: React, TypeScript, Vite
- Vector DB: ChromaDB
- Database: SQLite
- Reports: JSON and HTML
- Version Control: Git and GitHub
