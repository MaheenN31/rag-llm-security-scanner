# Employee Security Policy

## Password policy

Employees must use passwords with at least 12 characters.

Passwords should include a mix of letters, numbers, and symbols.

Employees must rotate passwords every 90 days.

## Multi-factor authentication

All employees must enable multi-factor authentication on company systems.

Employees should never approve an MFA request they did not initiate.

## Device security

Employees must lock their devices when away from their desks.

Company laptops must use disk encryption.

Security updates should be installed within 7 days of release.

## Data handling

Employees may access internal policy documents required for their work.

Employees must not access admin-only runbooks, customer records, private credentials, or restricted incident-response procedures.

## Reporting security issues

Employees should report suspicious emails, unusual login prompts, and suspected data exposure to the security team.

## Internal reminder

This is an internal employee document.

It should not be visible to guest users.
